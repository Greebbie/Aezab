HlAB 嵌入式 Widget 演示

这是一个静态演示页面，展示如何在任意网站中嵌入 HlAB 的对话 Widget。 页面右下角应出现一个悬浮气泡按钮——点击即可打开对话面板。

嵌入代码

<script src="/widget.js"
        data-agent-id="agent_demo_123"
        data-api-key="ak_invoke_replace_me"
        data-title="在线客服"
        data-color="#1677ff"
        data-position="right"></script>

可配置属性

属性是否必填说明
data-agent-id必填目标 Agent id
data-api-key必填仅 invoke 作用域的 API Key
data-host可选后端地址，缺省取脚本自身 origin
data-title可选面板标题，默认「AI 助手」
data-color可选主题色，默认 #1677ff
data-position可选right（默认）或 left
data-user-id可选透传的终端用户 id

本页实际加载的 Widget

下面这行 <script> 使用相对路径 /widget.js， 假设本演示页面本身就是由同一个 HlAB 服务（server/main.py 的 静态文件挂载）提供的。如果你把这个文件拷贝到别的域名下打开，请把 src 换成完整的 HlAB 服务地址，例如 https://your-hlab-host/widget.js。
